## Description

ContextIQ brings AI capabilities directly to your browsing experience without switching tabs.

**KEY FEATURES:**

✨ **Summarize** - Get concise summaries of any selected text
✍️ **Rewrite** - Improve clarity and readability of your content
🌐 **Translate** - Translate text to any language instantly
💡 **Explain** - Understand complex concepts in simple terms
❓ **Ask Questions** - Get answers based on selected context

**HOW IT WORKS:**

1. Select any text on a webpage
2. Right-click to open context menu or use the floating toolbar
3. Choose an AI action (Summarize, Rewrite, Translate, Explain, or Ask)
4. View AI-generated results in the beautiful side panel
5. Copy results with one click

**WHY CONTEXTIQ?**

🚀 **Seamless Integration** - Works on any website, no tab switching required
⚡ **Fast & Efficient** - Get instant AI responses without leaving your page
🎨 **Beautiful UI** - Modern, clean interface with smooth animations
🔒 **Secure** - Your data is processed securely with proper rate limiting
💪 **Powerful** - Powered by Grok AI for accurate and helpful responses

**USE CASES:**

- Research: Quickly summarize long articles and papers
- Writing: Improve your content with AI-powered rewriting
- Learning: Get explanations of complex topics
- Communication: Translate content to any language
- Productivity: Ask questions about any text you're reading

**PRIVACY & SECURITY:**

- Secure API communication
- No permanent data storage
- Rate limiting for fair usage
- Open source backend available

**REQUIREMENTS:**

- Active internet connection
- Backend API access (setup instructions in documentation)
- Chrome browser version 114 or higher

**SUPPORT:**

Need help? Visit our GitHub repository for documentation, setup guides, and support.

## Permissions Justification

**contextMenus**
- Required to add right-click menu options for AI actions
- Allows users to quickly access Summarize, Rewrite, Translate, Explain, and Ask features

**activeTab**
- Required to read selected text from the current webpage
- Only accesses text when user explicitly selects and triggers an action

**sidePanel**
- Required to display AI-generated responses in Chrome's side panel
- Provides a clean, non-intrusive UI for viewing results

**storage**
- Required to save user preferences and settings
- All data stored locally in browser

**host_permissions (<all_urls>)**
- Required to work on any website
- Allows users to use AI features on any webpage they visit
- Only activates when user explicitly selects text and triggers an action
